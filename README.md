# FinAnalyst Pro

AI-powered financial analysis application.

## 🎯 Purpose
Analyze financial data and generate key insights, KPIs, and valuations automatically.

## ✨ Features
- 20+ financial KPIs (Gross Margin, EBITDA, ROE, ROA, etc.)
- True Value (Enterprise Value) calculation – the "elite" KPI
- Intelligent insights and trend detection
- Interactive charts and visualizations
- PDF report export
